# 交易执行入口

这个目录放“人会直接运行”的交易执行入口。

约定：

- 日常只记一个入口：
  - `运行交易日常.py`
- Windows 命令行/双击优先使用：
  - `运行交易日常.cmd`
- 根目录中的
  - `交易_一键入口.py`
  - `交易_半自动流程.py`
  - `交易_稳定性巡检.py`
  继续保留为兼容壳，避免旧命令失效。

建议：

- 盘前：`运行交易日常.cmd preopen 2026-04-08`
- 收盘后：`运行交易日常.cmd eod 2026-04-07`
- 盘后复核：`运行交易日常.cmd postcheck 2026-04-07`
- 查看状态：`运行交易日常.cmd status 2026-04-08`
