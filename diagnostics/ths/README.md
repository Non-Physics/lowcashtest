# 同花顺诊断脚本目录

这个目录用于收纳同花顺模拟盘相关的专项诊断工具。

当前这里已经是同花顺诊断脚本的真实归档位置。

兼容策略如下：

- `diagnostics/ths/*.py`
  作为规范入口
- 项目根目录下保留同名薄包装
  用于兼容旧命令习惯

真实实现文件目前使用 `*_impl.py` 命名，后续若结构继续稳定，可再考虑去掉这一层命名过渡。
