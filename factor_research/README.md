# 因子研究目录

这个目录服务于独立的“因子选股研究”支线。

当前只建立最小骨架，不直接影响现有交易主线。

建议后续由研究支线逐步补充：

- `data_access.py`
  复用项目现有本地数据读取能力
- `universe.py`
  定义研究股票池与过滤规则
- `factor_library.py`
  实现候选因子
- `labels.py`
  定义与主线时序兼容的收益标签
- `evaluator.py`
  做分组收益、RankIC、时间切分评估
- `reporting.py`
  输出研究汇总与诊断结果

建议配套根目录研究入口：

- `因子研究总览.py`
- `因子选股回测.py`
- `因子效果诊断.py`

建议运行输出目录：

- `outputs/因子研究/reports/`
- `outputs/因子研究/backtests/`
- `outputs/因子研究/factor_panels/`

重要边界：

- 不修改当前默认交易入口
- 不修改同花顺半自动交易主链
- 不把实验结果直接接入交易执行
- 所有结论必须相对于当前基线策略给出
